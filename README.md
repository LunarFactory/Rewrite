# Rewrite

이 프로젝트는 `Rewrite`의 Unity 클라이언트 애플리케이션입니다.

## 📂 프로젝트 구조

주요 에셋과 코드는 `Assets` 폴더 아래 규칙에 따라 정리되어 있습니다.

```text
Rewrite/
├── Assets/
│   ├── Prefabs/        # 재사용 가능한 게임 오브젝트 및 UI 프리팹 모음
│   ├── Scenes/         # 게임 화면 및 씬 (TitleScene 등)
│   ├── Scripts/        # C# 스크립트 소스 코드 (게임 로직 및 UI 컨트롤 등)
│   ├── Settings/       # URP 및 프로젝트 전역 설정, Input Actions 파일 등
│   └── Sprites/        # UI 리소스, 백그라운드, 캐릭터 등 이미지/텍스처 에셋
├── Packages/           # Unity 패키지 모듈 관리 폴더
└── ProjectSettings/    # Unity 에디터 및 프로젝트 글로벌 설정
```

*(참고: 루트 디렉토리에 있는 `.py` 스크립트들은 에셋 추출용 임시 스크립트 도구들입니다.)*

## 🚀 구동 방법

본 프로젝트는 **Unity 6000.4.1f1** 버전을 사용하여 개발되었습니다. 원활한 실행을 위해 해당 버전을 맞춰주시는 것을 권장합니다.

### 1. 필수 사전 준비
- [Unity Hub](https://unity.com/kr/download) 설치
- Unity 에디터 버전 **6000.4.1f1** 설치

### 2. 프로젝트 열기 및 실행
1. **Unity Hub 열기**: 컴퓨터에서 Unity Hub를 실행합니다.
2. **프로젝트 가져오기**: 상단 우측의 `Open` (혹은 `Add`) 버튼을 클릭하여, 이 기재된 `README.md` 파일이 위치한 `Rewrite` 최상위 폴더를 선택합니다.
3. **프로젝트 실행**: 리스트에 등록된 프로젝트를 클릭해 Unity 에디터를 실행합니다.
   *최초 실행 시 에셋 임포트 및 라이브러리 초기화로 인해 다소 시간이 걸릴 수 있습니다.*
4. **메인 씬 실행하기**: 
   - 하단 `Project` 창에서 `Assets/Scenes` 폴더로 진입합니다.
   - `TitleScene.unity` (메인 타이틀) 씬을 더블 클릭하여 엽니다.
   - 에디터 상단 정중앙의 **재생(Play) 버튼 ▶️**을 눌러 정상적으로 구동되는지 확인합니다.

---

> **Tip!** 실행 중 문제가 발생할 경우 Unity 에디터 하단의 `Console` 탭에서 에러 로그를 확인해주세요.
